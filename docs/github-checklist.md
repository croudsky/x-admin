# GitHub Checklist

## Initial Setup

- [ ] Repository description を設定
- [ ] Topics を設定
- [ ] `main` の branch protection を有効化
- [ ] `test-and-build` を required status checks に追加
- [ ] Force push を禁止
- [ ] Branch deletion を禁止
- [ ] PR review を必須化
- [ ] CODEOWNERS review を有効化

## Collaboration

- [ ] Labels を作成
- [ ] Project board を作成
- [ ] Issue template を確認
- [ ] PR template を確認

## Secrets / Environments

- [ ] `development` environment を作成
- [ ] `production` environment を作成
- [ ] 必要な secrets を登録
- [ ] 本番用 secret を開発用と分離

## Nice To Have

- [ ] Release 作成フローを決める
- [ ] Version tag 運用を決める
- [ ] Deploy workflow を追加する
- [ ] Dependabot を追加する
