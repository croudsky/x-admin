# Oku

X運用自動化アプリのモノレポです。

## Stack

- Next.js
- Nest.js
- Prisma
- PostgreSQL
- Docker Compose

## ディレクトリ

- `apps/web`: Next.js 管理画面
- `apps/api`: Nest.js API
- `packages/shared`: 共有型
- `prisma`: データモデルとマイグレーション管理
- `docs`: ドキュメント集約先
